# EXCEL DASHBOARD: Snitch Fashion Sales Analysis

## 📌 Project Overview
This dashboard provides a comprehensive analysis of sales performance for **Snitch**, a leading Indian D2C men's fashion brand. The project focuses on regional profitability and sales trends across major urban hubs.

## 📊 Key Metrics Tracked
*   **Total Profit:** Regional profitability analysis.
*   **Total Revenue:** Revenue trends across target cities.
*   **Average Discount:** Comparison between B2B and B2C segments.
*   **Total Sales Amount:** Breakdown of sales volume by city.

## 📁 Dataset Details
The analysis is based on the **Snitch Clothing Sales Dataset**, featuring:
*   **Cities:** Ahmedabad, Bangalore, Delhi, Hyderabad, Mumbai, and Pune.
*   **Segments:** B2B, B2C, and others.
*   **Transactions:** Approximately 2,500+ records.

## 🛠️ Tools Used
*   **Data Visualization:** Microsoft Excel 
*   **Data Source:** Snitch Fashion Sales Dataset
